# 🪨 STRATICA Project Summary

## Overview
STRATICA is a Physics-Informed AI framework for decoding Earth's 4.5-billion-year stratigraphic record through the Temporal Climate Integrity Index (TCI).

## Key Specifications
- **TCI Parameters**: 9 (LDR, ISO, MFA, MAG, GCH, PYS, VSI, TDM, CEC)
- **Validation Basins**: 47 across 6 continents
- **Classification Accuracy**: 96.2%
- **Isotope Back-cast RMSD**: 0.0018‰
- **Milankovitch Precision**: ±1,200 years

## Applications
- Paleoclimate reconstruction
- Mass extinction precursor detection
- Autonomous drill core analysis
- Earth System Sensitivity quantification
