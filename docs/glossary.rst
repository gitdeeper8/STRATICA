Glossary
========

.. glossary::

   TCI
      Temporal Climate Integrity Index - A composite metric integrating nine stratigraphic parameters to assess paleoclimate reconstruction quality.

   LDR
      Lithological Deposition Rate - Sediment accumulation rate parameter (20% weight).

   ISO
      Stable Isotope Fractionation - δ¹⁸O/δ¹³C paleothermometry parameter (15% weight).

   MFA
      Micro-Fossil Assemblage - Biostratigraphic age control parameter (12% weight).

   MAG
      Magnetic Susceptibility - Geomagnetic polarity reversals parameter (11% weight).

   GCH
      Geochemical Anomaly Index - Trace element event signatures parameter (10% weight).

   PYS
      Palynological Yield Score - Vegetation history parameter (9% weight).

   VSI
      Varve Sedimentary Integrity - Annual lamination preservation parameter (8% weight).

   TDM
      Thermal Diffusion Model - Burial depth and thermal maturity parameter (8% weight).

   CEC
      Cyclostratigraphic Energy Cycle - Milankovitch orbital frequencies parameter (7% weight).

   PETM
      Paleocene-Eocene Thermal Maximum - Rapid warming event ~55.9 Ma.

   Milankovitch Cycles
      Orbital forcing cycles affecting climate: eccentricity (100/405 kyr), obliquity (41 kyr), and precession (21 kyr).

   IODP
      International Ocean Discovery Program - Provides drill core data.

   PANGAEA
      Data publisher for Earth and environmental science.

   PINN
      Physics-Informed Neural Network - Neural network with physical constraints.
